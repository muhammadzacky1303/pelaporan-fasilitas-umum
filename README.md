# Aplikasi Pelaporan Fasilitas Umum

Aplikasi desktop interaktif yang bertujuan untuk mempermudah proses pelaporan dan pengelolaan data fasilitas umum. Aplikasi ini memungkinkan pengguna untuk mencatat laporan, menyimpan data secara lokal, serta menampilkan informasi laporan dalam bentuk tabel yang terstruktur dan mudah dipahami.

## Fitur Utama
- Menambahkan data laporan fasilitas umum
- Menampilkan data laporan dalam bentuk tabel
- Mengubah dan menghapus data laporan
- Penyimpanan data menggunakan database lokal

## Tujuan Aplikasi
Membantu pengguna dalam melakukan pelaporan fasilitas umum secara lebih efektif dan terorganisir.

## Teknologi yang Digunakan
- Python
- Framework GUI Desktop
- Database Supabase
